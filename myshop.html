Web Application with ASP.Net Core

About the Project

This project involved developing a robust web application using ASP.Net Core. The app focuses on real-time error handling, authentication, and API integrations.

  • Used scaffolding for authentication to streamline development.
  • Implemented Serilog for robust error handling and performance monitoring.
  • Utilized AAA (Arrange, Act, Assert) model for thorough testing.
  • Focused on stability and reliability by isolating application behaviors.
